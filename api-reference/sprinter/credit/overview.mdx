## Response Format

All transaction-building endpoints return unsigned calldata:

```json
{
  "calls": [
    {
      "chain": "8453",
      "to": "0x...",
      "data": "0x...",
      "value": "0"
    }
  ]
}
```

Sign and broadcast each `ContractCall` in the user's wallet. No API keys needed for on-chain operations.
